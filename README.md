# Ayed-step-academy-2026
الموقع الرسمي للتسجيل في دورة STEP المكثفة 2026 (أكاديمية عايد) — تجربة سريعة، نموذج تأكيد دفع، خطة مذاكرة حسب مستواك، ومحتوى محدث يواكب أحدث نماذج STEP. العرض الحالي 299 ريال حتى 29/01/2026 ثم يعود السعر الرسمي 449 ريال.

## التعديل السريع
- تاريخ نهاية الخصم: داخل `assets/js/app.js` في دالة `updateCountdown` (التاريخ 2026-01-29 23:59 بتوقيت السعودية).
- السعر الحالي والسعر الرسمي: داخل `index.html` في قسم `#pricing` و `#hero`.
- عدد المقاعد: داخل `assets/js/data.js` عبر المتغيرين `maxSeats` و`reservedSeats`.
- رسائل الشريط العلوي والتنبيهات: داخل `assets/js/data.js` في `tickerMessages` و`notifications`.
